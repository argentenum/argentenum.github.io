Cultural History of South Asia

Distant view of historical linkages
between South Asia and the world.

To be published as
“Visualizing the cultural history of South Asia”
in Maya Dodd and Nirmala Menon eds.
Practices of Digital Humanities in India: Learning by doing
(Digital Humanities Asia Series).
Taylor and Francis.

Scope

  • Collect from Wikidata information about all persons who were either born or died in any territory that is currently part of South Asia
  • Visualize them to study migration of Persons of Importance

Research Workflow

  • Data acquisition
  • Data cleaning
  • Data analysis
  • Data visualiztion
Listed according to the QS World University Rankings 2021-22

Data acquisition

  • Acquire necessary data from Wikidata using Wikidata query and R

Data cleaning

  • Check for integrity of the data
  • Dates and Location of Birth and Death are essential for completing the dataset
  • Occupation information can be optional

Tools used

  • Manual
  • Wikidata Query
  • R

Linked Data

Wikidata follows the
Resource Description Framework (RDF)

Knowledge Graph

  • RDF is a framework for representing data on the internet in an interconnected manner.
  • Represents the relationships between real world entities - persons, places, objects - as a network

Data analysis

  • Map migration by tracing Birth Location to Death Location
  • Analyse them according to Time
  • Tool used: R

Data visualization

  • Publish the results of the analysis to charts for easy comprehension

Tools used

  • R
  • Palladio
